# Docs

Папка `docs/` предназначена для документации проекта.

## Состав

- `SPECIFICATION.md` — спецификация проекта, сущности, роли, API, архитектура.
- `DEPLOYMENT_GUIDE.md` — инструкция по деплою на VPS.
- `SECURITY.md` — правила хранения секретов и защиты сервера.
- `API_TEST_PLAN.md` — план проверки API через Swagger.
- `screenshots/` — скриншоты для отчёта.
- `diagrams/` — диаграммы БД, use-case, архитектура.
- `instructions/` — Word/PDF-инструкции для запуска и сдачи.
